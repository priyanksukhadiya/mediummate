# MediumMate WordPress Theme

MediumMate is a clean, elegant and modern WordPress theme inspired by Medium's design aesthetics. Perfect for personal blogs, magazines, and professional publishing platforms.

## Features

- Clean and modern design
- Fully responsive layout
- Featured posts section
- Author bio and social links
- Related posts section
- Custom page templates
- Translation ready
- RTL support
- Bootstrap 4 framework
- Font Awesome 6 icons

## Installation

1. Navigate to Appearance > Themes in your WordPress dashboard
2. Click "Add New"
3. Click "Upload Theme"
4. Choose the mediummate.zip file
5. Click "Install Now"
6. Click "Activate"

## Requirements

- WordPress 5.0 or higher
- PHP 7.0 or higher

## Theme Options

The theme can be customized through the WordPress Customizer. Navigate to Appearance > Customize to access the following options:

- Site Identity (Logo, Site Title, Tagline)
- Colors
- Typography
- Layout Options
- Header Settings
- Footer Settings

## Menu Locations

The theme includes the following menu location:

- Primary Menu: Main navigation menu in the header

## Widget Areas

The theme includes the following widget area:

- Sidebar: Appears on the right side of pages and posts

## Templates

- `front-page.php`: Homepage template with featured posts
- `page-authors.php`: Custom template for displaying author list
- `single.php`: Single post template with author bio and related posts
- `archive.php`: Archive template for categories, tags, and dates
- `author.php`: Author archive template with bio and social links

## Development

### Prerequisites

- Node.js and npm (for asset compilation)
- Local WordPress development environment

### Building Assets

1. Navigate to the theme directory
2. Run `npm install` to install dependencies
3. Run `npm run build` to compile assets
