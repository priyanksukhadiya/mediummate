MediumMate is a clean, elegant and modern WordPress theme inspired by Medium's design aesthetics.

== Description ==

MediumMate is a beautiful WordPress theme designed for bloggers and writers who want a clean, modern, and professional-looking website. The theme is inspired by Medium's design aesthetics and focuses on typography and readability.

Key Features:

* Clean and modern design
* Fully responsive layout
* Featured posts section
* Author bio and social links
* Related posts section
* Custom page templates
* Translation ready
* RTL support
* Bootstrap 4 framework
* Font Awesome 6 icons

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

MediumMate works perfectly with most popular WordPress plugins. It has been tested with:
* Contact Form 7
* Yoast SEO
* WooCommerce
* Jetpack

= Can I customize the theme colors? =

Yes, you can customize colors through the WordPress Customizer.
